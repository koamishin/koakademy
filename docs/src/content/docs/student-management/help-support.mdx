If you need help, go to **Help** from the sidebar.

## Viewing your tickets

The Help page lists all support tickets you've created, with:
- Subject
- Type (Support / Issue / Suggestion)
- Status (Open / In Progress / Resolved)
- Priority (Low / Medium / High)
- Date created

## Creating a new ticket

1. Click **Create Ticket** or **New Ticket**.
2. Choose the **Type**:
   - **Support** — You need help with something
   - **Issue** — Something is broken or not working correctly
   - **Suggestion** — You have an idea for improvement
3. Enter a clear **Subject** (e.g. "Cannot view my grades for MATH101").
4. Write a detailed **Message** explaining your problem. Include steps you've already tried.
5. Set the **Priority**:
   - **Low** — Not urgent
   - **Medium** — Needs attention soon
   - **High** — Urgent, blocking your work
6. Attach files if needed (screenshots, documents — up to 10 MB each).
7. Click **Submit**.

## Replying to a ticket

1. Click any ticket in your list to open it.
2. Read the full conversation — your original message plus all replies from staff.
3. Type your reply in the message box at the bottom.
4. Attach files if needed.
5. Click **Reply**.

Staff members are notified when you create or reply to a ticket. You'll receive a notification when they respond.
