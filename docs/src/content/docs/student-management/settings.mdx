In the sidebar or settings area, you can select which **semester** and **school year** to view data for.

This affects pages like **Tuition** and **Classes** that show semester-specific information.

## How to switch

1. Open the settings dropdown (typically in the top-right corner or sidebar).
2. Select the **Semester** (e.g. "1st Semester" or "2nd Semester").
3. Select the **School Year** (e.g. "2025-2026").
4. Pages automatically reload with data from that period.

This is helpful when you want to:
- View tuition records from a past semester
- Check grades from a previous school year
- See your class history from earlier terms
