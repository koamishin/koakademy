## Customizing Your Class Appearance

1. Open the class detail page.
2. Click **Class Settings** (quick action button).
3. You can configure:

| Setting | What It Does |
|---|---|
| **Accent Color** | The color of the class card and header on your class list |
| **Background Color** | Background tint for the class page |
| **Banner Image** | A header image for the class page |
| **Enable Announcements** | Turn the class stream on or off for students |
| **Enable Grade Visibility** | Let students see their grades or hide them |
| **Allow Late Submissions** | Let students submit assignments after the due date |
| **Enable Discussion Board** | Allow students to discuss posts |
| **Start Date** | When the class officially begins |

4. Click **Save Settings**.

---

## Editing Class Schedules

1. Open the class detail page.
2. Click **Edit Class Details**.
3. In the schedules section, you can:
   - Add new schedule entries — day of the week, start time, end time, room
   - Edit existing entries
   - Delete entries
4. Click **Save**.

<Aside type="note">
Attendance sessions must match the schedule's day of the week. If you change a schedule day, update any existing attendance sessions accordingly.
</Aside>
