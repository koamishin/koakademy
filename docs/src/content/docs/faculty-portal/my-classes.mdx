## Viewing Your Class List

Each class appears as a colored card showing:

| What You See | What It Means |
|---|---|
| **Subject Code** (e.g., CC101) | The official course code |
| **Subject Title** | The full name of the subject |
| **Section** (e.g., BSIT-1A) | The class section you teach |
| **Students** | How many students are enrolled |
| **Room** | Where the class meets |
| **Schedule** | Days and times (e.g., "Mon/Wed 10:00-11:30") |
| **Card Color** | Each class can have its own accent color |

You can customize card colors from [Class Settings](/faculty-portal/class-settings).

## Creating a New Class

1. Click the **Create Class** button.
2. Choose the **Classification** — College or Senior High School.
3. Select the **Subject** (for college) or **SHS Strand and Subject** (for SHS).
4. Choose which **Courses** this class is offered to (for college: BSIT, BSHM, BSBA, etc.).
5. Enter a **Section** name (e.g., "BSIT-1A" or "G11-STEM-A").
6. Choose a **Room** from the available rooms.
7. Set the **Maximum Slots** — how many students can enroll.
8. Click **Create**.

The class is created with you as the assigned teacher, using the current semester and school year.

---

## Editing a Class

1. Open the class detail page.
2. Click **Edit Class Details** (quick action button).
3. You can change the room, assigned teacher (to transfer the class), maximum slots, section name, semester, school year, classification, strand, subject code, and schedules.
4. Click **Save**.

## Class Detail Page

Click any class card to open its full detail page. This is your command center for that class, with several tabs:

### Overview Tab
- **Metrics cards** — Enrolled Students (slots filled), Sessions per Week, Class Type (College or SHS), Room Assignment
- **Student roster** — Every student with name, student ID, email, status, and grades (prelim, midterm, finals, average)
- **Schedule** — Days, times, and room for every session
- **Attendance calendar** — Color-coded dots showing which days have attendance recorded
- **Quick actions** — Edit Class Details, Manage Students, Class Settings, Post Announcement

### Attendance Tab
Take and track attendance. See [Taking Attendance](/faculty-portal/attendance).

### Grades Tab
Enter and manage student grades. See [Recording Grades](/faculty-portal/grades).

### Stream Tab
Post announcements, assignments, quizzes, and activities. See [Class Posts](/faculty-portal/class-posts).

### Students Tab
Add, remove, and transfer students. See [Managing Students](/faculty-portal/managing-students).

### Schedule Tab
View and edit the weekly schedule grid for this class.

## Activity Log

The class detail page includes a **Recent Activity** section showing actions like:
- "Student [Name] enrolled in this class"
- "Post [Title] was created"
- "Attendance session for [Date] was recorded"
- "Grades were updated"

Each entry shows the action, who was involved, and when it happened. This gives you a running history of everything that happens in your class.
