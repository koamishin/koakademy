## What You See

The schedule shows all your classes arranged by day and time, Monday through Saturday. Each entry shows:
- Subject code and title
- Section
- Room
- Start and end times

This is a read-only view of your teaching load for the current semester. Use this page to plan your week and see where you need to be and when.
