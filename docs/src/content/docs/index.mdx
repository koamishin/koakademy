## Get Started in 60 Seconds

The fastest way to run KoAkademy is with our pre-built Docker image:

```bash "Terminal"
docker run -d \
  --name koakademy \
  -p 8000:8000 \
  -e APP_KEY=$(openssl rand -base64 32) \
  docker.io/yukazakiri/koakademy:latest
```

Then visit `http://localhost:8000`.
