**KoAkademy** is an open-source learning management system (LMS) built for schools, universities, and training centers. It combines a powerful admin panel, a student portal, and a developer-friendly API — all in one modular Laravel application.

<CardGrid stagger>
	<Card title="Admin Panel" icon="setting">

	</Card>
	<Card title="Student Portal" icon="open-book">
		Inertia + React frontend with course enrollment, schedules, grades, and payment tracking.
	</Card>
	<Card title="REST API" icon="seti:json">

	</Card>

## Architecture Overview

KoAkademy is a full-stack Laravel application with the following stack:

| Layer | Technology |
|---|---|
| **Backend** | Laravel 12, PHP 8.4, FrankenPHP (Octane) |
| **Frontend** | React 19, Inertia.js, Tailwind CSS v4, shadcn/ui |
| **Admin** | Filament v4, Livewire v3 |
| **Database** | PostgreSQL 17 (primary), SQLite (local), MySQL 8.4 |
| **Cache/Queue** | Redis |
| **Search** | Meilisearch |
| **Queue Worker** | Laravel Horizon |
| **Monitoring** | Laravel Pulse, Laravel Telescope |
| **PDF** | Cloudflare Browser Rendering / DOMPDF |
| **Mail** | Mailpit (dev), SMTP (production) |

## Who Is This For?

- **Schools & Universities** — Manage students, courses, enrollments, grades, and finances.
- **Training Centers** — Track learner progress, issue certificates, and handle payments.
- **Developers** — Extend via modules, integrate via API, or white-label the platform.

## Self-Host or Contribute?

KoAkademy is built for two kinds of people. Pick the path that fits you:

### I want to self-host KoAkademy

You are running KoAkademy for your school, university, or training center. You want a stable, production-ready deployment with minimal setup.

**Recommended paths:**

| Path | Best For | Time |
|---|---|---|
| [Docker Compose](../docker/#docker-compose) | Production or self-hosted | 5 minutes |
| [Docker Run](../docker/#quick-run) | Quick evaluation | 1 minute |
| [Laravel Herd](../installation/#laravel-herd) | macOS / Windows — zero-config | 5 minutes |

After deploying, read [Configuration](../configuration) to set up domains, mail, and optional services.

### I want to develop or contribute

You are a developer who wants to extend KoAkademy, fix bugs, or build new modules. You need a local development environment with hot-reload, debugging, and test tooling.

**Recommended paths:**

| Path | Best For | Time |
|---|---|---|
| [Sail (Docker)](../installation/#docker--laravel-sail) | Any OS — fully isolated dev stack | 10 minutes |
| [Laravel Herd](../installation/#laravel-herd) | macOS / Windows — fastest non-Docker path | 5 minutes |
| [Native Setup](../installation/#native-manual) | Linux / full control | 15 minutes |
| [Quick PHP Server](../installation/#quick-php-server) | Just try it out | 2 minutes |

After setting up, read [Development](../../development) for the daily contributor workflow, testing conventions, and how to submit pull requests.
